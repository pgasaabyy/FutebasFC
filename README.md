# ⚽ FutebasFC

Site de escalação de futebol desenvolvido como projeto de estudo em HTML e CSS.

## Sobre o projeto

O FutebasFC é uma interface web estática que simula a visualização de escalação de um time de futebol. O foco do projeto foi a estruturação semântica do HTML e a organização visual com CS, tendo a interface interativa com ajuda do JavaScript

## O que foi praticado

- Estruturação semântica com HTML5
- Estilização e posicionamento com CSS3
- Layout responsivo
